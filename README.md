# Site Oficial da Liga Desportiva de Aracimã

Este projeto é a interface web da Liga Desportiva de Aracinã, desenvolvida para exibir classificações, jogos e estatísticas dos campeonatos municipais.

## 🚀 Tecnologias Utilizadas
- **Angular 21** (Framework Frontend)
- **Angular Material** (Componentes de UI e Tabelas)
- **TypeScript** (Lógica e Tipagem)
- **C# / .NET 10** (Integração com API Backend)
- https://github.com/3emeart/LDABack

## 📌 Funcionalidades
- [x] Tabela de classificação dinâmica consumindo API real.
- [x] Cálculo automático de pontos e saldo de gols.
- [x] Design responsivo para dispositivos móveis.

## ⚙️ Como rodar o projeto
1. Certifique-se de ter o [Node.js](https://nodejs.org/) instalado.
2. Clone o repositório.
3. No terminal, execute `npm install`.
4. Rode com `ng serve` e acesse `http://localhost:4200`.
